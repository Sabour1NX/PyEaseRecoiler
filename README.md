## 项目介绍
PyEaseRecoiler是一款基于Python+Windows API开发的鼠标后坐力补偿工具，提供「纯脚本简单模式」和「复杂多阶段模式」两种操作方式：
- 纯脚本版（PyEaseRecoiler_pureEase.py）：轻量无冗余，仅保留核心单参数补偿功能，无复杂模式入口；
- 完整版（PyEaseRecoiler.exe）：支持简单/复杂模式切换，复杂模式可通过CompEditor编辑器配置多阶段补偿参数。

工具核心为模拟鼠标反向移动实现“后坐力补偿”，但**仅适用于技术验证，非游戏作弊工具**。

## 核心功能
✅ **纯脚本版核心特性**：
- 单组参数配置：自定义鼠标补偿距离、角度、启动延迟、触发间隔；
- F9热键快捷控制：一键启停，无需切换窗口；
- 配置自动保存：参数存入`recoil_config.json`，下次启动自动加载；
- 轻量化：无多余组件，仅依赖pywin32，启动速度快；
- 容错处理：缺失配置文件时自动生成默认参数，不影响启动。

✅ **通用特性**：
- 随机间隔偏移：模拟真人操作，减少机械性特征；
- 实时鼠标监听：仅在按住左键时触发补偿，松开即停止；
- 管理员权限检测：未获取权限时自动提示并退出。

## 环境要求
- 操作系统：Windows 7/10/11（32/64位）（其他系统完全不兼容）；
- Python版本：3.7+（需安装完整版本，包含tkinter组件）；
- 权限：必须拥有Windows管理员权限（非管理员运行会直接报错）；
- 依赖：仅需pywin32（Windows系统级鼠标控制）。

使用方法（纯脚本版，推荐）
1. 运行程序
将PyEaseRecoiler_pureEase.py保存到本地任意文件夹；
右键点击该文件 → 选择「以管理员身份运行」（双击直接运行会提示权限不足并退出）；
启动后会自动加载上次保存的配置，若无配置则使用默认参数（距离 10、角度 90°、延迟 100ms）。
2. 参数配置
界面滑块 / 输入框对应参数说明（所有参数实时生效，无需重启）：
参数名称	取值范围	功能说明
鼠标移动距离	0~100 像素	每次补偿的鼠标反向移动距离（值越大，补偿效果越强）
移动角度	0~360°	补偿方向（90°= 向上，270°= 向下，0°= 向右，180°= 向左）
启动延迟	0~1000 毫秒	按住鼠标左键后，延迟多久开始触发补偿（避免误触）
连续间隔	0~500 毫秒	多次补偿动作之间的基础时间间隔（值越小，补偿越密集）
随机间隔	0~100 毫秒	在连续间隔基础上增加的随机偏移量（模拟真人操作，减少检测风险）
3. 启停控制
启动：点击界面「启动」按钮，或按下键盘 F9 键（状态栏显示「运行中 - 按 F9 停止」）；
停止：再次按下 F9 键，或点击「停止」按钮（状态栏显示「已停止」）；
触发逻辑：启动后，仅在按住鼠标左键时触发补偿移动，松开左键立即停止。
4. 配置保存 / 读取
自动保存：修改参数后，点击「保存配置」或关闭程序时，参数会自动写入recoil_config.json（与脚本同目录）；
手动修改：可直接编辑recoil_config.json文件，格式如下（修改后重启程序生效）：

##json
{
  "distance": 10,
  "angle": 90,
  "delay": 100,
  "interval": 50,
  "random_interval": 10
}


**常见问题**
Q1：运行提示「权限不足，无法调用鼠标 API」？
A：必须右键脚本 → 选择「以管理员身份运行」，这是 Windows 系统限制，无管理员权限无法控制鼠标底层操作。
Q2：F9 热键无反应？
A：① 确认程序以管理员身份运行；② 检查是否有其他软件（如游戏、输入法）占用 F9 热键；③ 关闭杀毒软件 / 防火墙（可能拦截热键监听）。
Q3：《三角洲行动》使用后被封 7 天，能申诉吗？
A：不能！游戏厂商明确将此类工具判定为作弊，封禁处罚无法申诉，所有损失由您自行承担（详见免责声明）。
Q4：参数调整后无补偿效果？
A：① 确认已按下 F9 启动功能（状态栏显示「运行中」）；② 移动距离设为 0 时无效果，建议先设为 10 测试；③ 角度设置错误（如想向下补偿却设为 90°）。
Q5：程序启动后闪退？
A：① 检查 Python 版本是否≥3.7；② 重新安装 pywin32 依赖；③ 确认系统为 Windows（macOS/Linux 会直接闪退）。
