## 项目介绍
PyEaseRecoiler是一款基于Python+Windows API开发的鼠标后坐力补偿工具，提供「纯脚本简单模式」和「复杂多阶段模式」两种操作方式：
- 纯脚本版（PyEaseRecoiler_pureEase.py）：轻量无冗余，仅保留核心单参数补偿功能，无复杂模式入口；
- 完整版（PyEaseRecoiler.exe）：支持简单/复杂模式切换，复杂模式可通过CompEditor编辑器配置多阶段补偿参数。

工具核心为模拟鼠标反向移动实现“后坐力补偿”，但**仅适用于技术验证，非游戏作弊工具**。

## 核心功能
✅ **纯脚本版核心特性**：
- 单组参数配置：自定义鼠标补偿距离、角度、启动延迟、触发间隔；
- F9热键快捷控制：一键启停，无需切换窗口；
- 配置自动保存：参数存入`recoil_config.json`，下次启动自动加载；
- 轻量化：无多余组件，仅依赖pywin32，启动速度快；
- 容错处理：缺失配置文件时自动生成默认参数，不影响启动。

✅ **通用特性**：
- 随机间隔偏移：模拟真人操作，减少机械性特征；
- 实时鼠标监听：仅在按住左键时触发补偿，松开即停止；
- 管理员权限检测：未获取权限时自动提示并退出。

## 环境要求
- 操作系统：Windows 7/10/11（32/64位）（其他系统完全不兼容）；
- Python版本：3.7+（需安装完整版本，包含tkinter组件）；
- 权限：必须拥有Windows管理员权限（非管理员运行会直接报错）；
- 依赖：仅需pywin32（Windows系统级鼠标控制）。

## 使用方法（纯脚本版，推荐）
### 1. 运行程序
1. 将`PyEaseRecoiler_pureEase.py`文件保存到本地任意文件夹（建议单独创建文件夹，方便管理配置文件）；
2. 右键点击该`.py`文件 → 选择「以管理员身份运行」（⚠️ 双击直接运行会提示“权限不足”并退出，必须管理员权限）；
3. 程序启动后，会自动加载上次保存的参数配置；若首次运行（无配置文件），则默认加载以下参数：
   - 鼠标移动距离：10像素
   - 移动角度：90°（向上）
   - 启动延迟：100毫秒
   - 连续间隔：50毫秒
   - 随机间隔：10毫秒

### 2. 参数配置
程序界面包含「滑块」和「输入框」两种调节方式，所有参数修改后实时生效，无需重启程序：
| 参数名称       | 调节方式                | 取值范围       | 功能说明                                                                 |
|----------------|-------------------------|----------------|--------------------------------------------------------------------------|
| 鼠标移动距离   | 滑块+输入框（整数）     | 0~100 像素     | 每次补偿时鼠标反向移动的距离，值越大补偿效果越强（建议从10开始测试）     |
| 移动角度       | 滑块+输入框（整数）     | 0~360°         | 补偿方向定义：90°=向上、270°=向下、0°=向右、180°=向左（按屏幕坐标系）   |
| 启动延迟       | 滑块+输入框（整数）     | 0~1000 毫秒    | 按住鼠标左键后，延迟多久启动补偿（设50~200ms可避免误触）                 |
| 连续间隔       | 滑块+输入框（整数）     | 0~500 毫秒     | 多次补偿动作之间的基础时间间隔，值越小补偿越密集（最小可设0ms）           |
| 随机间隔       | 滑块+输入框（整数）     | 0~100 毫秒     | 在“连续间隔”基础上增加随机偏移，模拟真人操作（建议设10~30ms）             |

### 3. 启停控制
#### 启动方式（二选一）
- 界面操作：点击程序主界面的「启动」按钮，按钮文本会变为「停止」，状态栏显示「运行中 - 按F9停止」；
- 热键操作：直接按下键盘「F9」键，无需切换到程序窗口（热键全局生效）。

#### 停止方式（二选一）
- 界面操作：点击「停止」按钮，按钮文本恢复为「启动」，状态栏显示「已停止」；
- 热键操作：再次按下「F9」键，无论当前焦点在哪个窗口，都会立即停止补偿。

#### 触发逻辑
- 启动后，**仅在按住鼠标左键时**触发补偿移动，松开左键则立即暂停；
- 再次按住左键，会延续之前的参数继续补偿，无需重新启动程序。

### 4. 配置保存与读取
#### 自动保存
- 修改参数后，点击「保存配置」按钮，或直接关闭程序窗口，当前参数会自动写入`recoil_config.json`文件（与`.py`脚本同目录）；
- 下次启动程序时，会自动读取该配置文件，无需重新调节参数。

#### 手动修改
- 找到`recoil_config.json`文件，用记事本/VS Code等工具打开；
- 按照以下格式修改参数（需为整数，且在取值范围内），保存后重启程序生效：
```json
{
  "distance": 10,
  "angle": 90,
  "delay": 100,
  "interval": 50,
  "random_interval": 10
}
```

#### 恢复默认配置
- 删除`recoil_config.json`文件，再次启动程序，会自动生成默认配置文件。

## 常见问题
### Q1：运行提示「权限不足，无法调用鼠标API」？
A：必须右键`.py`脚本 → 选择「以管理员身份运行」，这是Windows系统对鼠标底层操作的权限限制，无管理员权限无法正常工作。

### Q2：F9热键无反应？
A：① 先确认程序已以管理员身份运行（非管理员状态热键监听会被系统拦截）；② 检查是否有其他软件（如游戏、输入法、快捷键工具）占用F9键；③ 暂时关闭杀毒软件/防火墙（部分安全软件会拦截热键监听）。

### Q3：《三角洲行动》使用后被封7天，能申诉吗？
A：不能！游戏厂商明确将此类工具判定为作弊，封禁处罚属于系统自动判定，无申诉渠道，所有损失由您自行承担（详见免责声明）。

### Q4：参数调整后无补偿效果？
A：① 确认已启动程序（按钮显示「停止」或状态栏显示「运行中」）；② 检查「鼠标移动距离」是否设为0（设为0则无移动），建议先设为10测试；③ 确认「移动角度」设置正确（如想向下补偿却设为90°，会向上移动导致无效果）；④ 检查是否按住鼠标左键（仅按住左键时才触发）。

### Q5：程序启动后闪退？
A：① 检查Python版本是否≥3.7（低版本不兼容tkinter和pywin32）；② 重新安装pywin32依赖（执行`pip uninstall pywin32`后再重新安装）；③ 确认系统是Windows（macOS/Linux无win32api，会直接闪退）；④ 若仍闪退，删除`recoil_config.json`文件后重试。
